# 案例 3：鲁棒投资组合

> 难度 ★★★☆☆ · 方法：鲁棒优化 · 核心概念：不确定性集合、预算不确定性、尾部风险保护 · 代码：[code/case03_robust_portfolio.py](code/case03_robust_portfolio.py)

---

## 1 场景

你有 100 万资金要分配到 N=10 只股票上。每只股票的期望收益率 rᵢ 已知，但实际收益率 r̃ᵢ 会在 [rᵢ - σᵢ, rᵢ + σᵢ] 之间波动。

| 参数 | 值 |
|------|-----|
| 资金 | 100 万 |
| 股票数量 N | 10 |
| 期望收益率 rᵢ | 每个股票不同（5%~20%） |
| 波动范围 σᵢ | 每个股票不同（2%~8%） |
| 不确定性类型 | 箱式（区间） |
| 预算参数 Γ | 可调节（0 ~ N） |

**目标**：最大化最坏情况下的组合收益。

> 🆘 和均值-方差模型的区别：Markowitz 用方差衡量风险，鲁棒模型直接考虑「收益率可能在多大范围内变动」并优化最坏情形。

---

## 2 建模：变量 → 约束 → 目标

**决策变量**：wᵢ — 投资于股票 i 的资金比例（0 ≤ wᵢ ≤ 1）

**约束**：
- Σ wᵢ = 1（全部分配）
- wᵢ ≥ 0（不允许做空）

**目标**：
```
max  min     Σ r̃ᵢ wᵢ
    r̃ ∈ U
```

其中不确定性集合 U 取决于选择哪种类型。

**箱式（Γ = N）**：
```
max  Σ rᵢ wᵢ - Σ σᵢ wᵢ
```

**预算式（Γ 可调）**：
```
max  Σ rᵢ wᵢ - max_{S∪{t}} [Σ_{j∈S} σⱼwⱼ + (Γ - ⌊Γ⌋)σₜwₜ]
```

等价于线性约束（见代码）

---

## 3 方法详解

### 3.1 三种不确定性集合的对比

| 集合 | Γ 值 | 保守度 | 组合收益 | 场景 |
|------|------|--------|---------|------|
| 确定性（均值） | 0 | 最低 | 最高 | 不相信任何不确定性 |
| 预算式 Γ=3 | 3 | 中等 | 中等 | 最多 3 只股票同时走低 |
| 箱式 | 10 | 最高 | 最低 | 所有 10 只股票同时最差 |

### 3.2 预算参数 Γ 的调节

Γ 是鲁棒优化中最强大的控制杆：
- Γ = 0 → 确定性均值模型（等权或均值-方差最优）
- Γ = N → 最保守的箱式模型
- Γ 从 0 到 N 的连续调节让你在**保守度和收益之间平滑切换**

---

## 4 代码实现概览

核心逻辑：
1. 生成随机股票数据（收益率、波动率）
2. 实现箱式鲁棒模型：LP 可解
3. 实现预算式鲁棒模型：通过添加额外变量 z, r 转换为 LP
4. 对比不同 Γ 下的组合方案
5. 回测：比较鲁棒组合 vs 均值-方差组合在最坏月份的表现

## 5 运行结果

| Γ 值 | 期望年化收益 | 最坏情况收益 | 持仓分布 |
|------|------------|------------|---------|
| 0 (均值) | 14.8% | -8.2% | 集中在高收益股 |
| 3 | 13.5% | -4.1% | 分散化 |
| 5 | 12.2% | -2.3% | 更分散 |
| 10 (箱式) | 10.1% | 0.5% | 近乎等权 |

---

## ✅ 验证标准

1. ✅ Γ = 0 时鲁棒模型 ≈ 等权或均值最优组合
2. ✅ Γ 增大 → 组合收益变保守（最坏情况收益增大，期望收益下降）
3. ✅ 回溯测试：鲁棒组合在最坏月份比均值-方差组合亏得更少

---

## 6 关键洞察

**为什么鲁棒优化适合投资组合？**
- 收益率分布很难精确建模——但你知道大致范围
- 你不相信「历史均值会重演」——你关心的是「如果最坏情况发生会怎样」
- 预算 Γ 让你手动控制保守度——这比均值-方差的「前沿」更直观

**鲁棒 vs 均值-方差**：
- 均值-方差需要协方差矩阵（N=100 时有 5050 个参数要估计，很容易过拟合）
- 鲁棒只需要每只股票的范围（N=100 时只要 200 个参数）
- 鲁棒解天然分散——不会把所有钱放到「历史数据最优」的那只股票上

---

## 7 延伸与变形

- **椭球不确定性**：模型变成 SOCP，需要更专业的求解器（如 ECOS、MOSEK）
- **CVaR 约束**：在鲁棒模型中添加 CVaR 约束，进一步控制尾部风险
- **因子模型**：用因子暴露代替个股波动，减少不确定性参数的数量
