# 案例 3：鲁棒投资组合优化

> 难度 ★★★☆☆ · 鲁棒优化 · 核心概念：不确定性集合、预算不确定性、最坏情况保护 · 代码：`code/case03_robust_portfolio.py`

---

## 1 场景

你有 **100 万资金**要分配到 N=10 只股票上。

每只股票你查了一下历史数据，有个「预期收益率」r_i。但你也知道——**预测不靠谱**。实际收益率 r̃_i 可能在 [r_i - σ_i, r_i + σ_i] 之间波动。

常规做法是用马科维茨均值-方差模型：最大化「收益 - λ×风险」。但均值-方差有个致命的毛病——它对预期收益率的输入极其敏感。把某只股票的预期收益率从 8% 改到 8.1%，最优配置可能从 10% 变成 0%。

**问题**：你怎么分配这 100 万，才能确保**最坏情况下也不至于亏太多**？

---

## 2 建模

### 变量

- w_i：投资于股票 i 的比例（i = 1, ..., 10）
- Σ w_i = 1, w_i ≥ 0

### 不确定性

实际收益率 r̃_i 在**箱式不确定性集合**中：

```
U = { r̃_i ∈ [r_i - σ_i, r_i + σ_i], i = 1,...,10 }
```

也就是说，每只股票的收益率可能比预期低最多 σ_i，也可能高最多 σ_i——但我们只关心最坏情况，所以「低」的方向更重要。

### 目标：最坏情况下的最大收益

```
max_w   min_{r̃ ∈ U}  Σ_i  r̃_i · w_i
```

内层 min 是最坏情况的收益率（WMV：Worst-Case Mean-Variance）。给定 w，最坏情况就是每只股票都取最低收益率 r_i - σ_i：

```
最坏收益 = Σ_i (r_i - σ_i) · w_i = Σ_i r_i · w_i - Σ_i σ_i · w_i
```

所以鲁棒投资组合模型等价于：

```
max   Σ_i r_i · w_i - Σ_i σ_i · w_i
s.t.  Σ_i w_i = 1,  w_i ≥ 0
```

> **"把所有 σ_i·w_i 加起来从期望收益里扣掉"**——这就是最坏情况保护。每只股票的风险溢价（σ_i）变成对你的组合收益的「惩罚」。

### 预算不确定性（进阶版）

上面的箱式假设**所有股票同时表现最差**——这太保守了。现实中不太可能 10 只股票在同一天同时跌到最狠。

预算不确定性引入一个参数 Γ（Gamma）：

- Γ = 0：没有股票会表现最差 → 等价于均值模型
- Γ = N（=10）：所有股票同时最差 → 等价于箱式模型
- Γ = 3：最多 3 只股票会同时达到最差收益率（其余取预期值）

```
max   Σ_i r_i · w_i - max_{S: |S|≤Γ} Σ_{i∈S} σ_i · w_i
```

这个 max 选的是让惩罚最大的 Γ 只股票——也就是最坏的情况。

### 三种集合对比

| 集合类型 | 数学形式 | 保守度 | 计算代价 | 一句话 |
|---------|---------|-------|---------|--------|
| 箱式 (Box) | Σ(r_i - σ_i)w_i | ⭐最高 | LP | 所有股票一起崩 |
| 预算 (Budget) | Σr_i w_i - max_{|S|≤Γ} Σ_{i∈S} σ_i w_i | 可控 | LP | 最多 Γ 只崩 |
| 椭球 (Ellipsoidal) | μᵀw - κ√(wᵀΣw) | ⭐⭐中等 | SOCP | 越偏离越不可能 |

## 3 方法详解

### 箱式鲁棒

直接求解线性规划：

```
max   Σ (r_i - σ_i) w_i
s.t.  Σ w_i = 1,  w_i ≥ 0
```

这会把几乎所有资金都投到「最坏情况收益率最高」的那只股票上——也就是 r_i - σ_i 最大的那只。结果通常是非常**集中**的。

### 预算鲁棒

用 Bertsimas-Sim 方法转换为 LP：

```
max   Σ r_i w_i - z·Γ - Σ p_i
s.t.  z + p_i ≥ σ_i w_i,  ∀i
      z, p_i ≥ 0
      Σ w_i = 1,  w_i ≥ 0
```

这个转化通过一组线性约束完美刻画了「最多 Γ 个最坏情况」的约束。当 Γ 从 0 逐渐增加到 N，组合从分散走向集中。

### 椭球鲁棒

用二阶锥规划（SOCP）：

```
max   μᵀw - κ √(wᵀΣw)
s.t.  Σ w_i = 1,  w_i ≥ 0
```

其中 κ 控制椭球的大小。κ=0 → 均值模型，κ 越大越保守。

import cvxpy as cp
import numpy as np

np.random.seed(42)
n = 10
mu = np.random.uniform(0.05, 0.15, n)      # 预期收益率
sigma = np.random.uniform(0.05, 0.20, n)   # 波动幅度

# === 箱式鲁棒 ===
w_box = cp.Variable(n)
prob_box = cp.Problem(cp.Maximize(mu.T @ w_box - sigma.T @ w_box),
                      [cp.sum(w_box) == 1, w_box >= 0])
prob_box.solve()

# === 预算鲁棒 (Gamma=3) ===
Gamma = 3
w_bgt = cp.Variable(n)
z = cp.Variable()
p = cp.Variable(n)
prob_bgt = cp.Problem(
    cp.Maximize(mu.T @ w_bgt - Gamma*z - cp.sum(p)),
    [cp.sum(w_bgt) == 1, w_bgt >= 0,
     z + p >= cp.multiply(sigma, w_bgt), z >= 0, p >= 0])
prob_bgt.solve()

## 6 关键洞察

1. **鲁棒优化不是「预测」，而是「防护」**。你不需要猜哪只股票收益率最高，你只需要知道它们各自波动的范围。这在金融里特别实用——因为收益率预测误差极大，但波动范围相对可估计。

2. **箱式鲁棒过度集中**。因为它只看 r_i - σ_i，排名第一的股票会吃掉几乎所有权重——这不是分散化，是换了一种赌博。

3. **预算不确定性的参数 Γ 是个旋钮**。你可以根据自己的风险偏好调节它。Γ=3 意味着「我接受最多 3 只股票同时崩盘」——这是一种设计师可控的保守度。

4. **鲁棒组合天然抵抗「garbage in, garbage out」**。均值-方差模型对参数极度敏感，而鲁棒模型因为内置了「防护罩」，参数小幅变化不会导致组合剧烈变动。

## 7 延伸与变形

1. **数据驱动的不确定集**——不是拍脑袋设 σ_i，而是用历史数据的协方差结构构建椭球或预算集。

2. **结合 CVaR 约束**——在鲁棒模型中加入 CVaR ≤ threshold 的约束，双保险。

3. **多期鲁棒组合**——考虑交易成本和再平衡的多周期版本。

4. **与 Black-Litterman 结合**——先用 BL 模型得到「更靠谱的预期收益率」，再用鲁棒优化保护估计误差。
