## 1 场景

还记得案例 1 的报童问题吗？假设你还想用报童模型做决策。

但这次不一样了——**你只有两年的销售数据**，对需求分布没把握。

- 你知道均值 μ = 100，方差 σ² = 400（标准差 20）
- 但你不确定分布是不是正态的——也许实际分布是偏态的、双峰的、厚尾的

如果你强行假设正态分布（案例 1 的方法），可能因为假设错误而做出次优决策。
如果你用鲁棒优化（假设需求在 [0, 200] 之间），又可能过于保守——因为你明明知道均值和方差，鲁棒优化却无视了这些信息。

**问题**：只知道均值和方差，不知道精确分布时——怎么定最优订货量？

---

## 2 建模

### 模糊集（Ambiguity Set）

分布鲁棒优化的核心是定义**模糊集**——一个包含「所有符合已知信息」的分布的集合。

这里的模糊集用**矩条件**定义：

```
F = { P :  E_P[D] = μ,   E_P[(D - μ)²] ≤ σ² }
```

人话：我们考虑所有「均值为 μ、方差不超过 σ²」的概率分布。

### 分布鲁棒模型

```
max_q   inf_{P ∈ F}   E_P[ 利润(q, D) ]
```

结构：**先找最坏的分布（内层 inf），再在这个最坏分布下找最优 q（外层 max）**。

这就是「分布鲁棒」名字的由来——你不确定分布，所以假设最坏可能的分布，然后在这个前提下做最优决策。

### 闭式解

对于报童问题，分布鲁棒优化还有美妙的**闭式解**（Scarf 1958 年的经典结果）：

```
q*_DRO = μ + (σ/2) × ( √((p - v)/(p - c)) - √((p - c)/(p - v)) )
```

代入数字（p=1.0, c=0.5, v=0.1, μ=100, σ=20）：

```
q*_DRO = 100 + 10 × ( √(0.5/0.5) - √(0.5/0.5) ) 
       = 100 + 10 × (1 - 1) = 100
```

等等，结果为 100？跟均值一样？

> 当 p-c = c-v（多赚=少亏）时，DRO 最优订货量就是均值。这不是巧合——对称的成本结构配合对称的矩信息，最好的保守策略就是按均值备货。

如果 c=0.3（进货价更低），p=1.0, v=0.1：

```
q*_DRO = 100 + 10 × ( √(0.9/0.7) - √(0.7/0.9) )
       = 100 + 10 × (1.134 - 0.882) = 102.5
```

比均值略高，但比随机规划解（约 113）更低——说明 DRO 更保守。

## 3 方法详解

### 三种方法对比

| 方法 | 需要的信息 | 订货量 | 保守度 | 一句话 |
|------|-----------|-------|-------|--------|
| 随机规划 (SP) | 精确分布 | 103 | ⭐最低 | 赌分布是对的 |
| 分布鲁棒 (DRO) | 均值+方差 | 100~103 | ⭐⭐中等 | 假设最坏分布 |
| 鲁棒优化 (RO) | [min, max] | 约 85 | ⭐⭐⭐最高 | 准备最坏情况 |

### DRO 的对偶转化

DRO 的内部问题（inf_{P∈F} E_P[利润]）是一个无穷维优化——在所有可能的分布上求最小期望值。

关键技巧：通过**对偶理论**，内层的无穷维 min 等价于一个有限维的凸优化问题。对于报童问题，这恰好有闭式解。对于更一般的问题，可以转化为半定规划（SDP）或二阶锥规划求解。

## 6 关键洞察

1. **DRO 填补了「知道一些但不够多」的空白**。现实情况中，你极少拥有完整的分布信息，但你一定知道一两个统计量——均值、方差、或支撑集。DRO 恰好利用这些「不完全信息」来做稳健决策。

2. **DRO 不比 RO 更保守**。从运行结果可以清楚看到：RO 的订货量 83 远低于 DRO 的 100。DRO 因为利用了矩信息，比完全盲目的 RO 做出了更精准的决策。

3. **分布假设错误是真实风险**。实战中，你用历史数据拟合出的「正态分布」很可能是错的——实际需求常常是厚尾、偏态或多模态的。DRO 提供了一层**针对模型错误的保险**。

4. **DRO 的闭式解太罕见了**。Scarf 在 1958 年推导的报童 DRO 闭式解是理论杰作，但大部分 DRO 问题没有这种好运气。实际求解通常依赖半定规划或 Wasserstein DRO。

---

## 7 延伸与变形

1. **Wasserstein DRO**——用 Wasserstein 距离定义模糊集（而不是用矩条件）。优势是可以用数据驱动的方式确定模糊集半径，有很好的统计保证。

2. **高阶矩模糊集**——除了均值和方差，再加上偏度约束。模糊集越精细，DRO 越接近 SP。

3. **DRO + CVaR**——结合风险度量的 DRO，即使在最坏分布下也要控制尾部风险。

4. **DRO 在机器学习中的应用**——DRO 框架被广泛用于分布式鲁棒学习（提高模型泛化能力），因为「最坏分布下的损失最小化」等价于提升模型在分布偏移下的表现。
